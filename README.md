# dig&drop (仮) — 縦スクロール採掘アクション（TypeScript + HTML5 Canvas）
このゲームは、ドリルを操作して地中深く掘り進んでいく **「縦スクロール採掘アクションゲーム」** です。  
迫りくる画面スクロールに追いつかれないように、ひたすら下へ下へと掘り進み、到達深度（m）＝スコアを競います。
---
## 🎮 ゲーム概要
- **目的**：スクロールに巻き込まれないように掘り進み、より深い地点（m）を目指す
- **スコア**：潜った距離（メートル）を表示。深いほどランキング上位
---
## 🕹 操作方法
- **← / →**：左右移動
- **↓**：加速（ドリルドライブ）
※キーボード操作前提（PC向け）
---
## ⛏ 地形
- **土 (Dirt)**：掘れる（進める）
- **岩 (Rock)**：硬くて掘れない（避ける必要あり）
掘れる場所を瞬時に見つけて、下へ潜っていく判断が重要です。
---
## 💀 ゲームオーバー条件
- **画面上端のスクロールに巻き込まれたらアウト**
- **HP（バッテリー）が尽きたらアウト**
---
## ✨ 特徴的な要素
### 1) モグラ（敵キャラ）
- 地中を横切るように妨害してきます
- ぶつかるとダメージ
- 見た目は **2倍サイズの巨大でかわいいモグラ**
### 2) バッテリー（回復アイテム）
- 土の中に埋まっている **緑色のバッテリー** を取るとHP回復
- こちらも **超巨大サイズ** で表示され、見逃しにくい設計
### 3) フィーバーモード（無敵）
- 特定条件（例：連続で掘る など）で発動する無敵モード
- 発動中は **金色のオーラ** をまとい、一気に掘り進める爽快感を狙います
---
## 🖼 ビジュアル / 世界観

- 世界観：リアルな土と岩の質感 + メカニカルなバッテリーで採掘現場の雰囲気
